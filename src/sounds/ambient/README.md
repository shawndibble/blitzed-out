# Ambient Music Audio Files

This directory contains ambient soundscape audio files for the game.

## Required Files

| File Name    | Description                      | Target Size |
| ------------ | -------------------------------- | ----------- |
| lounge.mp3   | Calm, sophisticated lounge music | <300KB      |
| intimate.mp3 | Soft, romantic ambient music     | <300KB      |
| party.mp3    | Upbeat, energetic party music    | <300KB      |

## Audio Requirements

- **Format**: MP3 (best browser compatibility)
- **Duration**: 30-60 seconds (will loop seamlessly)
- **File Size**: Under 300KB each
- **Quality**: 128kbps is sufficient for ambient background

## Royalty-Free Sources

The following sites offer free, royalty-free music suitable for this use:

- [Pixabay](https://pixabay.com/music/) - Free for commercial use, no attribution required
- [Mixkit](https://mixkit.co/free-stock-music/) - Free music, no attribution required
- [Free Music Archive](https://freemusicarchive.org/) - Various licenses, check individual tracks

## Audio Processing Tips

1. Trim to 30-60 seconds for a seamless loop point
2. Apply fade-in/out at loop boundaries
3. Normalize audio levels
4. Export at 128kbps MP3 for optimal size/quality
5. Test looping to ensure smooth transitions
